## 🧭 What weclaw does

weclaw helps you connect to agents through WeChat ClawBot. It is made for people who want a simple way to start and use the app on Windows.

Use it to:
- connect to WeChat-based agent tools
- work with ClawBot and OpenClaw setups
- link to agent services from a desktop app
- keep the setup in one place

## 🖥️ What you need

Before you start, check these basics on your Windows PC:

- Windows 10 or Windows 11
- 4 GB RAM or more
- 200 MB free disk space
- An internet connection
- A WeChat account if your setup needs one
- Permission to run downloaded apps

If your PC is older, the app can still work, but it may start slower.

## 🛠️ First-time setup

When you open weclaw for the first time, you may need to set a few options:

- choose your language if asked
- sign in to WeChat if the app asks for it
- allow network access when Windows asks
- enter your agent address or token if your setup uses one
- save your settings before you close the app

If you are not sure what to enter, use the values from the person or team that gave you the app.

## 🔗 Connect to an agent

To connect a new agent:

1. Open weclaw
2. Go to the connection screen
3. Enter the agent details
4. Save the settings
5. Start the connection
6. Check the status line for a success message

If the connection works, the app should show that the agent is online or ready.

## 🔍 If something does not work

Try these steps if the app does not open or connect:

- make sure you downloaded the full file
- run the file again as an administrator
- check that your internet is working
- close the app and open it again
- restart your PC
- check that Windows did not block the file
- confirm that the agent address is correct
- confirm that your login details are correct

If the app still does not work, download the file again from the project page and try once more.
